아주 유용한 슬라이싱

2021-06-02

슬라이싱 이란?슬라이싱(Slicing)은 연속적인 타입(문자열, 리스트, 튜플 등) 의 데이터를 조작하는 방법이다. 슬라이싱을 이용하면 그러한 연속적인 타입의 데이터를 범위를 지정해 잘라서 사용할 수 있다. 그야말로, 햄을 슬라이스해서 일부분만 사용하는 것처럼 데이터를 원하는 두께로 슬라이스 한다. S &#x3D; &quot;안녕하세요&quot; i &#x3D; S[1:3] # i &#x3D; &quot;녕하&quot; 인덱싱은?인덱싱(Indexing)도 슬라이싱과 비슷하게 연속적인 타입의 데이터를 조작하는 방법의 하나이지만, 슬라이싱과는 다르게 범위를 지정해 가져오는 것이 아니라 하나의 요소만을 지정해서 가져오거나 사용..

리트코드 561.Array Partition I

2021-05-31

문제

문제 원문

Given an integer array nums of 2n integers, group these integers into n pairs (a1, b1), (a2, b2), ..., (an, bn) such that the sum of min(ai, bi) for all i is maximized. Return the maximized sum.

nums라는 길이 2n의 숫자 리스트가 주어졌을 때, 임의의 n개의 페어 (a1, b1), (a2, b2), ...., (an, bn) 를 구성하고 그 페어들의 최솟값을 전부 더한 값의 최댓값을 구하는 문제

일단 문제 말부터가 어려웠다. 입출력 예를 확인해보자.

입출력 예시

Ex1

Input: nums = [1,4,3,2]
Output : 4
Optimal pairing: (1,2), (4,3)

Ex2

Input: nums = [6,2,6,5,1,2,]
Output: 9
Optimal pairing: (2,1), (2,5), (6,6)

풀이

나의 풀이

1. Input List = nums
2. sort(nums)
3. for(i=0; ;i += 2)
4. &nbsp;&nbsp;&nbsp;&nbsp;sum += min(nums(i), nums(i+1))

나의 코드

class Solution:
    def arrayPairSum(self, nums: List[int]) -> int:
        sum = 0
        i = 0
        nums.sort()
        while i < len(nums)-1 :
            sum += min(nums[i], nums[i+1]) 
            i += 2
        return sum

Runtime: 288 ms

페어끼리 묶어서 최솟값을 구했을 때, 작은값과 큰 값 사이에 차가 크지 않아야 최솟값의 합이 최댓값이 될 것이라고 생각해서 인풋 리스트를 오름차순으로 정렬한 후에 앞에서부터 두 개씩 묶어서 최솟값을 구하고 그것들을 다 더해주었다.

생각보다 간단하게 해결가능 했다.
하지만, 실행시간에 조금은 아쉬움이 있었다. 효율 좋은 해답을 확인해보자

해답 코드

class Solution:
    def arrayPairSum(self, nums: List[int]) -> int:
        return sum(sorted(nums)[::2])

Runtime: 248 ms

sorted() 와 슬라이스를 이용해서 아주 간단하게 해결했다.

개인적 코멘트 슬라이스를 쓰는 버릇을 들여야겠다. 파이썬 좋다는게 이런건데 전혀 파이썬스러움을 이용하지 못하는 것 같다.

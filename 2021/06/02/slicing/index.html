아주 유용한 슬라이싱

2021-06-02

슬라이싱 이란?

슬라이싱(Slicing)은 연속적인 타입(문자열, 리스트, 튜플 등) 의 데이터를 조작하는 방법이다. 슬라이싱을 이용하면 그러한 연속적인 타입의 데이터를 범위를 지정해 잘라서 사용할 수 있다. 그야말로, 햄을 슬라이스해서 일부분만 사용하는 것처럼 데이터를 원하는 두께로 슬라이스 한다. 

S = "안녕하세요"
i = S[1:3] 
# i = "녕하"

인덱싱은?

인덱싱(Indexing)도 슬라이싱과 비슷하게 연속적인 타입의 데이터를 조작하는 방법의 하나이지만, 슬라이싱과는 다르게 범위를 지정해 가져오는 것이 아니라 하나의 요소만을 지정해서 가져오거나 사용한다. 

S = "안녕하세요"
i = S[0]
# u = "안"

리스트를 이용해 봤으면 이러한 사용방식은 이름은 몰라도 아주 익숙할 것이다. 그리고, 마찬가지로 리스트를 이용해 봤으면 알 수 있겠지만 문자열의 맨 처음 시작이 ‘0’부터 시작하게 된다.

그리고, 음수값을 사용하면 뒤에서 부터 인덱싱이 이루어진다. 여기서는 ‘-1’ 부터 시작하기 때문에 주의해야한다. 

S = "안녕하세요"
i = S[-3]
# i = "하"

슬라이싱 사용법

문자열이든 리스트이든 튜플이던간에 어떤 연속적인 타입의 데이터 S가 있다고 했을 때, 슬라이싱은 기본적으로

S[start:end:step]

의 형태로 사용할 수 있다.

Start로는어디부터 자르기를 시작할 지를, End로는 어디까지 자를지를을 정한다. 아무것도 입력하지 않으면 디폴트 값으로 Start는 처음부터로, End는 맨 끝까지로 설정된다.

여기서 주의해야될 것은 End 위치에 있는 요소는 포함이 안된다는 점이다. 예를 들어, 

i = S[1:3]

이라고 했을 때, i에 S[3]은 포함되지 않는다.

Step은 몇 개씩 끊어서 요소를 가져올 지를 정한다.예를 들어, step이 2라면 시작지점부터 2칸씩 이동한 후에 값을 가져온다. step이 음수일 경우엔, 뒤에서부터 슬라이싱이 이루어진다. 이때의 Start와 End 값은 각각 뒤의 인덱스와 앞의 인덱스값이 들어가야한다. 

S = [1,2,3,4,5]
i = S[::2]
# i = [1,3,5]

슬라이싱 예시

S = [1,2,3,4,5]
  1. 인덱스1에서 인덱스 4이전까지 슬라이싱
S[1:4] #[2,3,4]
  1. 인데스 3에서 끝까지 슬라이싱
S[3:] #[4,5]
  1. 처음부터 인덱스 3이전까지 슬라이싱
S[:3] #[1,2,3]
  1. 처음부터 끝까지 슬라이싱(참조가 아닌 사본 리턴)
S[:] #[1,2,3,4,5]
  1. 인덱스 1에서 뒤에서 2번째 이전까지 슬라이싱
S[1:-2] #[2,3]
  1. 뒤에서 5번째 부터 뒤에서 2번째 이전까지 슬라이싱
S[-5:-2] #[1,2,3]
  1. 처음부터 끝까지 2개씩 슬라이싱
S[::2] #[1,3,5]
  1. 인덱스 4에서 인덱스 1까지 거꾸로 슬라이싱
S[4:1:-1] #[5,4,3]
  1. 전체를 뒤집는다
S[::-1] #[5,4,3,2,1]
